1. Project2 directory should be in the user directory, e.g. '/home/012/y/yx/yxc180006/Project2'.


2. config file should be in the user directory, e.g. '/home/012/y/yx/yxc180006/config.txt'.


3. To start the project, run the command:
   >> ./launcher.sh

   In the 'Project2' directory, each server's printout lines will be saved as "*.out".


4. To run the verification, run the command:
   >> ./verify.sh

   After it is finished, five files containing calculated data and sorted time interval will be generated at where the config file is put. Also the server's printout lines will be saved as "verify.out".


5. To clean up, run the command:
   >> ./cleanall.sh
